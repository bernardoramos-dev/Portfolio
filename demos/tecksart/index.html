Conceito

Cada peça começa como uma questão, não uma solução. O processo começa quando encontramos a pergunta certa - uma forma que ainda não existe, uma função que merece uma nova linguagem visual.

02
02 / 03

Precisão

O design é refinado até que cada ângulo seja intencional. Não existe detalhe decorativo - se está ali, tem razão de estar. A forma final é aquela que não pode ser simplificada sem perder a essência.

03
03 / 03

Objeto

O resultado é uma peça que existe no espaço de forma completa - não precisa de contexto, de legenda, de explicação. Ela simplesmente está ali, e o espaço ao redor responde a ela.
